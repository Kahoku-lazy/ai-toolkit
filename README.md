# AI工具包

一个实用的 AI 工具集合项目。

## 项目简介

本项目旨在收集和整理各种实用的 AI 工具和功能。

## 开始使用

```bash
# 克隆项目
git clone https://github.com/Kahoku-lazy/ai-toolkit.git

# 进入目录
cd ai-toolkit
```

## 目录结构

| 目录 | 说明 |
|------|------|
| `Agent/` | 存放子代理设定 |
| `Agent Skills/` | 存放 Skills 技能 |
| `Markdown/` | 存放定向业务、RAG 所需的相关文档 |
| `Notes/` | 存放笔记文件 |
