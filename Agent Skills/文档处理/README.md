# 文档处理 Skills

此目录包含用于文档处理和创建的 Agent Skills。

## 包含的 Skills

| Skill | 功能描述 | 文件类型 |
|-------|----------|----------|
| **frontend-slides** | 创建动画丰富的 HTML 演示文稿 | HTML/PPT |

## 计划添加的 Skills

- **docx** - Word 文档处理
- **xlsx** - Excel 电子表格处理
- **pptx** - PowerPoint 演示文稿处理
- **XMind** - 思维导图处理
- **pdf** - PDF 文档处理

## 使用说明

每个 Skill 都有独立的 `SKILL.md` 文件，包含详细的使用方法。
激活 Skill 后，AI 会自动读取该文件并提供相应功能。

## 目录结构

```
文档处理/
├── README.md              # 本文件
├── frontend-slides/       # HTML 演示文稿技能
│   ├── SKILL.md
│   ├── STYLE_PRESETS.md
│   ├── scripts/
│   └── ...
└── [其他 skills]/
```
